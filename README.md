# AI Automation Journey — Real Estate Finance

Building a job-ready portfolio for a career as an **AI & Automation Specialist in Real Estate Finance**.

This repository documents practical projects that combine:

- Real estate and mortgage finance
- Python and SQL
- Data analysis and Power BI
- AI / LLM applications
- Workflow automation
- Azure cloud services

## Portfolio Roadmap

### Project 1 — DSCR Investment Property Analyzer
A Python application that evaluates rental-property financing scenarios using NOI, debt service, DSCR, LTV, cap rate, cash flow, and basic sensitivity analysis.

**Skills:** Python, financial calculations, data validation, Streamlit (planned), testing.

### Project 2 — AI Mortgage Document Analyzer
Extract structured information from loan documents and summarize key underwriting fields with an LLM-assisted workflow.

**Skills:** Python, OCR/document intelligence, Azure AI, structured extraction, prompt engineering.

### Project 3 — Real Estate Finance RAG Assistant
A retrieval-augmented AI assistant that answers questions from approved lending guidelines and finance documents with source citations.

**Skills:** embeddings, vector search, RAG, LangChain/LlamaIndex, Azure OpenAI.

### Project 4 — Automated Loan Pipeline Intelligence
Analyze loan-pipeline data, flag bottlenecks, generate status summaries, and automate operational alerts.

**Skills:** Python, SQL, Power BI, Power Automate, APIs.

### Project 5 — Commercial Real Estate Deal Analyzer
Underwrite CRE acquisitions using NOI, cap rate, debt yield, DSCR, LTV, IRR, and scenario analysis.

**Skills:** finance modeling, Python, pandas, visualization, AI-generated investment summaries.

### Project 6 — AI Real Estate Finance Copilot
Capstone application combining document ingestion, deal analysis, retrieval, and workflow automation into one finance-focused AI assistant.

**Skills:** Azure, LLMs, RAG, agents, APIs, automation, deployment.
