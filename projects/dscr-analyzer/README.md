# Project 1 — DSCR Investment Property Analyzer

A practical Python project for analyzing rental-property financing scenarios.

## Why This Project Matters

DSCR lending is a strong example of the intersection between real estate finance and technology. This project demonstrates the ability to translate a finance workflow into working software.

## Version 1 Features

- Calculates annual gross rental income
- Calculates operating expenses
- Calculates net operating income (NOI)
- Calculates amortizing loan debt service
- Calculates debt-service coverage ratio (DSCR)
- Calculates loan-to-value ratio (LTV)
- Calculates capitalization rate
- Calculates estimated annual cash flow

## Run Locally

```bash
python app.py
```

Enter a hypothetical deal when prompted.
